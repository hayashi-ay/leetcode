初見では解けず、ヒントを見つつ書いた。

> Hint: Use recursion. If root.val <= V, you split root.right into two halves, then join it's left half back on root.right.

```python
class Solution:
    def splitBST(self, root: Optional[TreeNode], target: int) -> List[Optional[TreeNode]]:
        
        if root is None:
            return [None, None]
        
        # rootがtarget以下 -> 右側を分割する必要がある
        if root.val <= target:
            smaller, larger = self.splitBST(root.right, target)
            root.right = smaller
            return [root, larger]
        
        # rootがtargetより大きい -> 左側を分割する必要がある
        smaller, larger = self.splitBST(root.left, target)
        root.left = larger
        return [smaller, root]
```
