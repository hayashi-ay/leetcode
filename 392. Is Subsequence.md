sとtの両方を前から見ていく。両者が同じ場合はどちらも進める、異なる場合はtのみを進める。tの探索が終わるまでにsの探索が終わればOK。

1st
```python
class Solution:
    def isSubsequence(self, s: str, t: str) -> bool:
        si = 0
        ti = 0

        while si < len(s) and ti < len(t):
            if s[si] == t[ti]:
                si += 1
                ti += 1
            else:
                ti += 1
        return si == len(s)
```

2nd

基本的には1stと同じ考え方だが、メインのループではtのみが回る。
```python
class Solution:
    def isSubsequence(self, s: str, t: str) -> bool:
        if s == "":
            return True
        si = 0
        for ti in range(len(t)):
            if s[si] != t[ti]:
                continue
            si += 1
            if si == len(s):
                return True
        return False
```
