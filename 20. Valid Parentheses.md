
プッシュダウンオートマトン。左括弧だとスタックに積む、右括弧だとスタックからポップして対になる括弧であれば次に進む。テープを読みきった際にスタックが空であれば受理状態。それ以外は受理しない。

1st

```python
class Solution:
    def isValid(self, s: str) -> bool:
        left_brackets = []
        pair = {"(": ")", "{": "}", "[": "]"}
        for c in s:
            if c in pair:
                left_brackets.append(c)
                continue
            if not left_brackets:
                return False
            left_bracket = left_brackets.pop()
            if pair[left_bracket] != c:
                return False
        return len(left_brackets) == 0
```

2nd

1stとはところどこ違う書き方で書いた。

```python
class Solution:
    def isValid(self, s: str) -> bool:
        stack = []
        parentheses = {"(": ")", "{": "}", "[": "]"}
        for c in s:
            if c in parentheses:
                stack.append(c)
                continue
            if stack == []:
                return False
            if parentheses[stack[-1]] != c:
                return False
            stack.pop()
        if stack:
            return False
        return True
```
