再帰とループのどちらでも解ける。要は最短経路を求めよということなのでBFSで解くのが１番効率が良い。

1st

BFS
leafノードの判定を`is_leaf_node`に切り出すのは好みの問題。

```python
class Solution:
    def minDepth(self, root: Optional[TreeNode]) -> int:
        def is_leaf_node(node):
            return node and node.left is None and node.right is None

        if root is None:
            return 0
        nodes = []
        nodes.append(root)
        depth = 0
        while nodes:
            depth += 1
            next_nodes = []
            for node in nodes:
                if is_leaf_node(node):
                    return depth
                if node.left:
                    next_nodes.append(node.left)
                if node.right:
                    next_nodes.append(node.right)
            nodes = next_nodes
        raise Exception("unreachable")
```

再帰
```python
class Solution:
    def minDepth(self, root: Optional[TreeNode]) -> int:
        if root is None:
            return 0
        if root.left is None and root.right is None:
            return 1
        min_depth = inf
        if root.left:
            min_depth = min(min_depth, self.minDepth(root.left))
        if root.right:
            min_depth = min(min_depth, self.minDepth(root.right))
        return min_depth + 1
```

2nd

BFSでqueueを使う方法で解いた。

```python
class Solution:
    def minDepth(self, root: Optional[TreeNode]) -> int:
        if root is None:
            return 0

        nodes = deque() # (node, depth)
        nodes.append((root, 1))
        while nodes:
            node, depth = nodes.popleft()
            if node.left is None and node.right is None:
                return depth
            if node.left:
                nodes.append((node.left, depth + 1))
            if node.right:
                nodes.append((node.right, depth + 1))
        raise Exception("unreachable")
```
